## Commands

Scaffold the project first (run once):
```bash
npm create vite@latest . -- --template react
npm install
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

After scaffolding:
```bash
npm run dev      # Dev server at localhost:5173
npm run build    # Production build (verify this passes before marking any task done)
npm run preview  # Preview production build
```

## Brand Identity

Colors extracted from `logoViana.svg` (verify full palette against `brandinform.docx`):
- Pink/coral: `#E24A71`, `#E45360`, `#E66554`
- Gold/orange: `#F7BC58`, `#F29D56`
- Dark text: `#1A1B21`

These must be injected as named tokens in `tailwind.config.js`. Never use generic fallback fonts (Inter, Roboto, Arial) вЂ” use only the fonts specified in `brandinform.docx`. Colors come from `brand-tokens.json` (generated by `01-brand-analyst`); never invent values.
